Customising a theme

set_theme 1. Generate CSS variable assignments and configuration settings to customize the Gantt chart theme according to user requirements and apply them to the chart. Prompt example: I want to customise the Gantt chart theme. Create a bright and warm theme for me. Also, increase the height of the rows and bars. Make the link width 4px.
2. Change the current theme. Prompt example: Make tasks a bit lighter.
reset_theme Reset the current Gantt theme by clearing all custom CSS variables (back to defaults). Prompt example: Reset the current theme.
